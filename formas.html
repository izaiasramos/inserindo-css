CSS INLINE: é adicionado na linha que deseja adicionar o css,alterando assim o html,mais especificamente,ele é adicionado na abertura da tag desejada.

tag h1 no html sem css: < h 1 >treine CSS< /h1 >

Veremos asseguir um exemplo de css inline:

< h1 style=color:red> treine CSS < /h1>

O css é inserido na abertura da tag < h 1 >

Treine CSS

CSS INTERNO:é declarado/colocado dentro do head,veja um exemplo asseguir:

  1. < head >
  2. < style >
  3. h1 {color:blue; fonte-size:15px}
  4. pre {
  5. color:green;
  6. fonte-size:30px;
  7. }
  8. < /style >
  9. < /head >

Dentro do style coloco e organizo meu css,da maneira que achar mais fácil e legível,primeiro coloco o nome da tag html que desejo alterar,depois abro e fecho colchetes {....} dentro do colchetes será desenvolvido o css,como vimos no exemplo acima.

CSS EXTERNO:No css externo,é feito uma pasta separada somente para o css,para vincular essa pasta á nossa pasta html,e para que o html leia e desenvolva tudo oque estiver na outra pasta,é feito uma linkagem dentro do head (configuração) do html,ou seja,é feito um link da sua pasta css para que o html localize e vincule tudo,ficaria assim:
< link rel="stylesheet" href="nome da pasta.css"/>
assim o html importa os dados da pasta externa para vincular tudo
